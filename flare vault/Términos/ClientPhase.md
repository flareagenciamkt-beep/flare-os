# ClientPhase

Fase **operativa** de un [[Client]] (`currentPhase`): `onboarding` → `estrategia` → `produccion` → `revision_interna` → `revision_cliente` → `publicacion` → `optimizacion` → `reporte_mensual`.

Describe en qué punto del ciclo de servicio mensual está la cuenta (no confundir con [[ClientStatus]], que es comercial, ni con [[IdeaStatus]], que es por pieza).
